# Creative Visual Prompt

This prompt describes the visual layer for a single `creative` slide.

## Goal

Create one light, funny, salon-related image that can carry a short joke.

## Visual Rules

- Format: `1080x1920`
- Keep the visual simple and readable.
- Prefer object-led or scene-led humor.
- Do not depend on a perfect portrait.
- Do not use a finished hairstyle as the core joke.

## Planner Modes

- `exact_salon_room` if the salon room itself supports the joke
- `exact_salon_closeup` if a tighter detail works better
- `neutral_nonhuman_object` if the joke is mostly about objects or tools

## Best Visual Subjects

- tools with personality
- products acting like characters
- an empty chair with a strong salon mood
- a table corner with a tiny funny contrast
- a mirror or workstation detail that supports the joke

## Do Not Show

- heavy beauty editorial styling
- portrait-centered humor
- cringey meme templates
- messy low-quality clutter
- obvious AI text artifacts

## Visual Tone

- playful
- light
- human
- salon-coded
- realistic enough to feel usable in Telegram or social media
