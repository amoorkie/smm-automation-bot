# Slider Visual Prompt

This prompt describes the visual layer for a short `slider` carousel.

## Goal

Create 3 to 5 vertical slides that support a simple instruction or guide.

## Visual Rules

- Format: `1080x1920`
- Keep each slide readable.
- Prefer one idea per slide.
- When color accents are useful, prefer the salon brand palette `#CE6BD6` and `#A21A92`.
- Use those two colors as restrained accents instead of random accent colors.
- Use a real salon feel or a clean object scene.
- Do not make the set look like a glossy fashion campaign.

## Planner Modes

- `exact_salon_room` for room-led slides
- `exact_salon_closeup` for detail-led instructional slides
- `neutral_nonhuman_object` for object-led instructional slides

## Best Visual Subjects

- care products
- tools and working corner
- hands of a master in a neutral process detail
- a simple before-use / after-use logic without a literal before-after claim
- towel, brush, comb, bottle, diffuser, heat tools

## Do Not Show

- client portraits as the main object
- hero hairstyle framing
- full collage overload
- busy typography
- fake infographic polish
- unrelated decorative clutter

## Visual Tone

- clean
- practical
- calm
- realistic
- easy to map to a step-by-step guide
