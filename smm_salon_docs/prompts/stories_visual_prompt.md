# Stories Visual Prompt

This prompt describes the visual layer for a single `stories` slide.

## Goal

Create one realistic vertical image that supports a short hair-care story.

## Visual Rules

- Format: `1080x1920`
- Keep the image simple.
- Make it feel like a phone photo or a clean real-world salon visual.
- Prefer a real object, a product setup, a working corner, or a small salon detail.
- Do not center a finished hairstyle.

## Planner Modes

- `exact_salon_room` for a recognizable room anchor
- `exact_salon_closeup` for a tighter detail inside the same room
- `neutral_nonhuman_object` for a product-only or object-only image

## Best Visual Subjects

- shampoo and conditioner grouping
- a single care product plus supporting tools
- heat-protection detail
- a mirror corner with a small routine setup
- a towel, brush, bottle, or comb

## Do Not Show

- busy infographic layouts
- full collage treatment
- portrait-first framing
- fashion editorial styling
- polished ad-banner lighting

## Visual Tone

- calm
- readable
- real
- slightly imperfect
- close to everyday salon life
